# StudySnap Pro

The ultimate YouTube study companion — snap frames, take rich notes, and organize everything directly from your browser. 100% local and private.

## Features
- **Video Snaps**: Capture video frames straight from YouTube into your notes.
- **Rich Editor**: Highlight, mark up screenshots, format text, and add tables.
- **Local Storage**: Every note is saved automatically to your device and is fully searchable. Completely private with no tracking.
- **Export**: Easily export notes to PDF, Word, or Markdown.

## Installation (Developer Mode)
1. Download or clone this repository.
2. Go to `chrome://extensions/` in Google Chrome.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked** and select this directory.
5. Open a YouTube video and click the **StudySnap Pro** button to start taking notes!
